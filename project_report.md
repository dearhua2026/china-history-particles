# 项目报告：中国历史人物粒子交互背景

## 项目概述
本项目实现了一个基于 `tsParticles` 的高度视觉化网页背景。通过“神经元连接”式的点线交互，展示了一系列中国历史人物的名字，象征着文明脉络的交织与传承。

## 技术实现
- **核心逻辑**：使用 `tsParticles` 引擎，配置 `text` 形状粒子。
- **视觉风格**：
    - 深色科技背景（Radial Gradient）。
    - 玻璃拟态（Glassmorphism）UI 组件。
    - 响应式布局，适配移动端。
- **人物名单**：包含秦始皇、苏轼、诸葛亮等约 50 位极具代表性的历史人物。

## 主要文件
1. `index.html`: 结构定义与资源加载。
2. `style.css`: 现代视觉样式与动画定义。
3. `app.js`: 粒子引擎初始化与动态配置。

## 交互说明
- **鼠标悬停**：触发 `grab` 模式，建立与周边的临时连接。
- **鼠标点击**：触发 `push` 模式，向空间中注入更多历史节点。
- **自动游走**：粒子以恒定低速在屏幕内反弹，模拟思维的流动。

## 验证结果
- 经浏览器子代理验证，页面渲染正常，无控制台报错。
- 视觉效果达到“高端、科技、动态”的预期目标。
